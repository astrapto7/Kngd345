 SiesfiToken (SIESFI)

An ERC-20 custom token for the Seismic Network testnet. Mintable (up to 1B cap) and burnable.

## Features
- Max supply: 1,000,000,000 SIESFI
- Mintable by contract owner
- Burnable by any token holder

### 1. Install dependencies
